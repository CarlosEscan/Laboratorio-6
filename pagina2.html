Bootstrab

Bootstrap es un framework CSS creado por Twitter en 2010 para estandarizar sus herramientas. Anteriormente conocido como Twitter Blueprint, se convirtió en código abierto en 2011 bajo el nombre de Bootstrap. Este framework combina CSS y JavaScript para estilizar los elementos de una página HTML, proporcionando interactividad a través de componentes como menús de navegación, controles de página y barras de progreso.

Su principal objetivo es facilitar la construcción de sitios web responsivos, diseñados para funcionar en diferentes dispositivos, como computadoras de escritorio, tablets y smartphones. Funciona mediante archivos CSS y JavaScript que asignan características específicas a los elementos de la página. Bootstrap ofrece características como diseño responsivo, utilizando elementos div y clases container para estructurar la página de manera adaptable.

Bootstrab tiene una basta biblioteca de componentes, estos son slgunos de los principales:

Alertas

Bootstrap permite una configuración simple y rápida de diferentes tipos de alertas, con colores específicos, según la situación. Para mostrarle al usuario un alerta que indique atención, por ejemplo, simplemente debemos usar .alert-danger y aparecerá un cuadro de texto con un fondo rojo: ¡Atención! ¡Cuidado mensaje de alerta!

Carrusel

Un componente ampliamente utilizado en Bootstrap es el Carrusel, una presentación de diapositivas, es decir, una herramienta que permite la visualización de imágenes de manera receptiva. También permite la inclusión de efectos especiales para la transición de imágenes y controles de visualización, como por ejemplo los indicadores de “siguiente” y “anterior”.

Barra de navegación

La NavBar (barra de navegación), que permite la construcción de un sistema de navegación sensible. Es posible configurar diferentes formas de presentar el menú, elegir entre posicionamiento lateral o superior y, también, definir una forma de visualización que se pueda extender o contraer. También es posible determinar cómo mostrar los enlaces del menú, que pueden ser en forma de botón, enlace, menú suspenso, entre otras configuraciones, para facilitar la implementación de la navegación del sitio.

El framework utiliza tres tipos de containers: Container, Container-fluid y Container-{breakpoint}, que determinan el ancho y la longitud de la página según el tamaño del dispositivo. Mediante la asignación de clases en el HTML, como "rounded-sm" para bordes redondeados, se aplican estilos y funcionalidades a los elementos de la página. En resumen, Bootstrap simplifica el desarrollo web al proporcionar herramientas para crear sitios responsivos y atractivos con facilidad.
