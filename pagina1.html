HTML 5

HTML5 es el estándar utilizado para definir la estructura, diseño y contenido de una página web. Aunque no es un lenguaje de programación, sino un código que organiza elementos como texto, imágenes y videos, su versión más reciente, HTML5, presenta diversas características y mejoras respecto a sus predecesores:

Introduce nuevas etiquetas semánticas que facilitan la estructura de los documentos HTML y mejoran la indexación en buscadores.

Permite la inclusión de audio y vídeo mediante etiquetas multimedia.

Integra gráficos vectoriales escalables (SVG) y elementos para dibujar con la etiqueta canvas.

Reajusta elementos comunes en las páginas web.

Ofrece la posibilidad de incluir fórmulas matemáticas a través de MathML.

HTML5 tiene múltiples ventajas, como mejorar la experiencia del usuario en navegadores populares, ser compatible con diferentes dispositivos, mejorar el posicionamiento en buscadores y admitir funcionalidades multimedia sin necesidad de plugins. Además, facilita la introducción de elementos complejos como dibujos 3D o mapas interactivos, permite desplegar subsecciones en resultados de búsqueda de Google, optimiza el tiempo de carga del sitio web y mejora la seguridad de las páginas que lo utilizan

Funciona mediante un protocolo de solicitud/respuesta llamado HTTP para la comunicación entre clientes y servidores. Utiliza WebSockets, una tecnología que emplea HTTP para establecer una comunicación ligera entre el cliente y el servidor. Un navegador compatible con HTML5 utiliza la API de JavaScript para abrir, iniciar y cerrar conexiones WebSocket, permitiendo una conversación fluida entre cliente y servidor durante una conexión HTML5.
